# WinVClipboard

یک مدیر کلیپ‌بورد سریع و بومی برای ویندوز؛ با ظاهر آشنای `Win + V`، تاریخچهٔ بزرگ‌تر، پین، دسته‌بندی و میانبرهای نوشتاری.

[English](README.en.md) · [تغییرات نسخه‌ها](CHANGELOG.md) · [دریافت آخرین نسخه](https://github.com/PouryaRajaei/WinVClipboard/releases/latest)

## چرا WinVClipboard؟

کلیپ‌بورد داخلی ویندوز تعداد محدودی آیتم نگه می‌دارد و امکانات سازمان‌دهی آن کم است. WinVClipboard همان تجربهٔ سریع `Win + V` را حفظ می‌کند، اما تا ۲۰۰۰ آیتم، پین‌های پایدار، دسته‌بندی آیکن‌دار، تصویر بندانگشتی و جایگزینی خودکار متن را در اختیار شما می‌گذارد.

## قابلیت‌ها

- بازشدن با `Win + V` و جلوگیری از نمایش هم‌زمان پنل پیش‌فرض ویندوز
- ذخیرهٔ حداکثر ۲۰۰۰ آیتم متنی، تصویری و فایل‌ها
- نمایش فوری thumbnail تصاویر و ذخیره‌سازی آن‌ها در پس‌زمینه
- پین‌کردن آیتم‌های مهم؛ موارد پین‌شده همیشه در انتهای فهرست قرار می‌گیرند
- فیلتر مستقل برای نمایش فقط آیتم‌های پین‌شده
- چسباندن سریع ۹ پین اول با `Ctrl + 1` تا `Ctrl + 9`
- دسته‌بندی پین‌ها با چیپ‌های آیکن‌دار و مجموعه‌ای از آیکن‌های Material Design
- ساخت، ویرایش و حذف دسته‌ها و تغییر آیکن هر دسته
- جست‌وجوی زنده در متن‌ها و نام فایل‌ها
- تشخیص خودکار جهت متن: فارسی و عربی RTL، انگلیسی LTR
- انتخاب با کلیدهای بالا و پایین و چسباندن با `Enter`
- چسباندن مستقیم با دابل‌کلیک در آخرین پنجرهٔ فعال، بدون تغییر وضعیت maximize آن
- بازشدن روی مانیتوری که نشانگر موس در آن قرار دارد
- پشتیبانی صحیح از چینش‌های چندمانیتوره و DPIهای متفاوت
- اجرای تک‌نمونه‌ای؛ از بازشدن چند نسخهٔ هم‌زمان جلوگیری می‌شود
- حالت اجرای مخفی هنگام راه‌اندازی ویندوز با آرگومان `--startup`
- رابط کاملاً دوزبانهٔ فارسی/English با تغییر آنی جهت RTL/LTR و ذخیرهٔ زبان انتخاب‌شده
- سه اندازهٔ کوچک، متوسط و بزرگ برای پنل با ذخیرهٔ انتخاب کاربر
- آیکن System Tray با دسترسی سریع به بازکردن، تنظیمات و خروج
- کنترل اجرای خودکار همراه ویندوز از داخل تنظیمات
- پنجرهٔ تنظیمات مستقل با بخش‌های عمومی، ظاهر، حریم خصوصی، پشتیبان‌گیری و درباره
- Backup/Restore تاریخچه، پین‌ها، دسته‌ها، میانبرهای نوشتاری و تنظیمات
- توقف موقت ثبت، عدم ذخیرهٔ تصویر، حذف خودکار و مستثناکردن برنامه‌های حساس
- پوسته‌های تیره، روشن و System؛ تنظیم ظرفیت تاریخچه و اندازهٔ thumbnail
- ضبط زندهٔ میانبر دلخواه نمایش پنل و انتخاب `Ctrl/Alt + 1…9` برای پین‌ها
- بررسی نسخهٔ جدید از GitHub Releases
- دانلود، نصب و اجرای خودکار نسخهٔ جدید بدون نیاز به خروج دستی کاربر
- ساخت خودکار x64/ARM64 با GitHub Actions و زیرساخت امضا و MSIX

## میانبرهای نوشتاری (Text Expansion)

می‌توانید یک trigger و متن جایگزین تعریف کنید؛ برای مثال:

```text
Shortcut:   /addr1
Description: تهران، خیابان ایکس
```

پس از تایپ `/addr1` در یک ورودی متنی، پیشنهاد کنار نشانگر تایپ و روی همان مانیتور ظاهر می‌شود. با `Tab` متن کوتاه به Description تبدیل و با `Esc` پیشنهاد لغو می‌شود. جایگزینی با ورودی Unicode انجام می‌شود و کلیپ‌بورد فعلی شما را تغییر نمی‌دهد.

## کنترل‌ها

| عمل | کلید یا روش |
|---|---|
| بازکردن پنل | `Win + V` |
| حرکت بین آیتم‌ها | `↑` / `↓` |
| چسباندن آیتم انتخاب‌شده | `Enter` |
| چسباندن فوری | دابل‌کلیک روی آیتم |
| بستن پنل | `Esc` |
| چسباندن پین‌های ۱ تا ۹ | `Ctrl + 1` … `Ctrl + 9` |
| پذیرفتن پیشنهاد نوشتاری | `Tab` |
| لغو پیشنهاد نوشتاری | `Esc` |
| خروج کامل | دکمهٔ «خروج» یا `Alt + F4` |

## نصب برای کاربران

1. از صفحهٔ [Releases](https://github.com/PouryaRajaei/WinVClipboard/releases) فایل متناسب با پردازنده را دریافت کنید:
   - `win-x64` برای بیشتر کامپیوترهای Intel و AMD
   - `win-arm64` برای دستگاه‌های Windows on ARM
2. فایل ZIP را استخراج کنید.
3. `WinVClipboard.exe` را اجرا کنید.

خروجی‌های Release به‌صورت self-contained منتشر می‌شوند؛ بنابراین نصب جداگانهٔ .NET برای آن‌ها لازم نیست.

> برنامه از keyboard hook سراسری برای مدیریت `Win + V` و میانبرهای نوشتاری استفاده می‌کند. بعضی ابزارهای امنیتی ممکن است در اولین اجرا تأیید کاربر بخواهند.

### اجرای خودکار با ویندوز

یک Shortcut از فایل اجرایی داخل پوشهٔ Startup بسازید و `--startup` را به انتهای Target اضافه کنید:

```text
"C:\Path\To\WinVClipboard.exe" --startup
```

برای بازکردن پوشهٔ Startup، `Win + R` را بزنید و `shell:startup` را وارد کنید.

## ساخت از سورس

پیش‌نیازها: Windows 10/11، [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0) و Git.

```powershell
git clone https://github.com/PouryaRajaei/WinVClipboard.git
cd WinVClipboard
dotnet restore
dotnet build -c Release
```

ساخت فایل اجرایی self-contained:

```powershell
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true
dotnet publish -c Release -r win-arm64 --self-contained true -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true
```

## محل ذخیرهٔ داده‌ها

تاریخچه، دسته‌ها و میانبرهای نوشتاری به‌صورت محلی در مسیر زیر ذخیره می‌شوند:

```text
%LOCALAPPDATA%\WinVClipboard
```

هیچ همگام‌سازی ابری یا ارسال داده‌ای در برنامه انجام نمی‌شود.
